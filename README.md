# Calculadora-EBAC-PYTHON
Curos EBAC primeiro módulo: Python, Linux e GitHub.

Abra o arquivo no colab e execute as células:
Calculadora.ipynb

Explicação do Código Python
O programa utiliza um laço de repetição para manter a calculadora em execução até que o usuário escolha a opção de sair.

Fluxo do Programa:
Exibe um menu de opções.
Solicita ao usuário a operação desejada.
Recebe dois números.

Executa a operação escolhida:
Soma (+)
Subtração (-)
Multiplicação (*)
Divisão (/)
Mostra o resultado.
Retorna ao menu principal.
Encerra quando a opção 5 - Sair é selecionada.

Conceitos Utilizados
Estruturas de repetição (while)
Estruturas condicionais (if, elif, else)
Entrada de dados com input()
Conversão de tipos com float()
Operações matemáticas básicas
Controle de fluxo do programa
